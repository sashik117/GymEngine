# Ручне Тестування

## Web Preview

Локальне web-прев'ю зручно для швидкої перевірки візуалу.

```powershell
cd D:\GymEngine\apps\mobile
..\..\tools\flutter.ps1 build web --debug --no-wasm-dry-run
cd build\web
python -m http.server 5173 --bind 127.0.0.1
```

Відкрити:

```text
http://127.0.0.1:5173
```

Web-прев'ю використовує SQLite у браузері. Якщо оновити сторінку, локальні дані можуть скинутися. Нативні збірки використовують файлову SQLite-базу.

## Основний Flow

1. Відкрити `БАЗА`.
2. Перевірити перемикач `УКР | ENG`: українська має бути дефолтною.
3. Натиснути `СТВОРИТИ ДЕНЬ`.
4. Ввести назву, наприклад `Верх Тіла`.
5. Налаштувати `ВІДПОЧИНОК, СЕК` і `ЧАС ПІДХОДУ, СЕК`.
6. Додати готову вправу через чип `+ Жим Лежачи`.
7. Додати свою вправу: заповнити `Назва вправи`, `Група мʼязів`, натиснути `+`.
8. Перевірити `ПОРЯДОК ДНЯ`: вправи можна рухати вгору/вниз або видаляти.
9. Натиснути `ЗБЕРЕГТИ ДЕНЬ`.
10. Перевірити, що день показує список вправ по порядку і таймери.
11. Натиснути `ПОЧАТИ ТРЕНУВАННЯ`.
12. Перевірити кнопку назад у верхньому лівому куті активної сесії.
13. Перевірити `ПЛАН СЕСІЇ`, блок `SMART HISTORY` і кнопку `ЗАПИСАТИ ПІДХІД`.
14. Записати підхід і перевірити неоновий таймер `ВІДПОЧИНОК`.
15. Перейти до наступної вправи через `НАСТУПНА ВПРАВА`.
16. Натиснути `ЗАВЕРШИТИ ТРЕНУВАННЯ`.
17. На `ПІДСУМОК` перевірити об'єм, кількість підходів і кращий оцінений 1ПМ.
18. Повернутися `НАЗАД ДО БАЗИ`.
19. Відкрити `ДВИГУН` і перевірити графік об'єму та `КАРТА МʼЯЗІВ`.
